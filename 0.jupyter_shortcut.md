VS Code Jupyter Notebook-এর সবচেয়ে useful shortcuts:

| কাজ                        | Shortcut        |
| -------------------------- | --------------- |
| নিচে নতুন cell             | `B`             |
| উপরে নতুন cell             | `A`             |
| Selected cell delete       | `D` `D`         |
| Cell → Markdown            | `M`             |
| Cell → Code                | `Y`             |
| Cell run + নিচে যাওয়া      | `Shift + Enter` |
| Cell run + একই cell-এ থাকা | `Ctrl + Enter`  |
| Cell run + নিচে নতুন cell  | `Alt + Enter`   |
| Edit mode                  | `Enter`         |
| Command mode               | `Esc`           |
| Cell copy                  | `C`             |
| Cell cut                   | `X`             |
| Cell paste below           | `V`             |
| Undo cell delete           | `Z`             |

**গুরুত্বপূর্ণ:** `A`, `B`, `DD`, `M`, `Y` কাজ করাতে আগে `Esc` চাপ দিয়ে **Command Mode**-এ যেতে হবে।

তুমি যদি **cell উপরে/নিচে move করার shortcut** জানতে চাও, VS Code-এ default keybinding version অনুযায়ী আলাদা হতে পারে; `Ctrl + Shift + P` → `Notebook: Move Cell Up/Down` দিয়ে command দেখতে পারো।
