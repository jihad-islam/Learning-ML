# VS Code Jupyter Notebook Shortcuts

### Cell তৈরি ও Delete

| **Cell তৈরি ও Delete** | **Shortcut** |
| ---------------------- | -----------: |
| নিচে নতুন Cell         |          `B` |
| উপরে নতুন Cell         |          `A` |
| Selected Cell Delete   |      `D` `D` |
| Deleted Cell Restore   |          `Z` |

### Cell Type পরিবর্তন

| **Cell Type**   | **Shortcut** |
| --------------- | -----------: |
| Cell → Markdown |          `M` |
| Cell → Code     |          `Y` |

### Cell Run

| **Cell Run**                |    **Shortcut** |
| --------------------------- | --------------: |
| Run করে নিচের Cell-এ যাওয়া  | `Shift + Enter` |
| Run করে একই Cell-এ থাকা     |  `Ctrl + Enter` |
| Run করে নিচে নতুন Cell তৈরি |   `Alt + Enter` |

### Mode পরিবর্তন

| **Mode**     | **Shortcut** |
| ------------ | -----------: |
| Edit Mode    |      `Enter` |
| Command Mode |        `Esc` |

### Cell Copy, Cut & Paste

| **Cell Operations** | **Shortcut** |
| ------------------- | -----------: |
| Cell Copy           |          `C` |
| Cell Cut            |          `X` |
| Cell Paste Below    |          `V` |
| Cell Paste Above    |  `Shift + V` |

### Code Folding

| **Code Folding**     |            **Shortcut** |
| -------------------- | ----------------------: |
| Current Block Fold   |      `Ctrl + Shift + [` |
| Current Block Unfold |      `Ctrl + Shift + ]` |
| Fold All             | `Ctrl + K` → `Ctrl + 0` |
| Unfold All           | `Ctrl + K` → `Ctrl + J` |

### Code Editing

| **Code Editing**    |       **Shortcut** |
| ------------------- | -----------------: |
| Comment / Uncomment |         `Ctrl + /` |
| Find                |         `Ctrl + F` |
| Replace             |         `Ctrl + H` |
| Undo                |         `Ctrl + Z` |
| Redo                | `Ctrl + Shift + Z` |
| Autocomplete        |     `Ctrl + Space` |
| Format Cell         |  `Shift + Alt + F` |

### VS Code Navigation

| **Navigation**  |       **Shortcut** |
| --------------- | -----------------: |
| Command Palette | `Ctrl + Shift + P` |
| Quick Open      |         `Ctrl + P` |

> [!IMPORTANT]
> `A`, `B`, `D D`, `M`, `Y`, `C`, `X`, `V`, `Z`-এর মতো single-key shortcut ব্যবহার করতে আগে `Esc` চেপে **Command Mode**-এ যেতে হবে।
